# Roadmap

## v0.1

- Provider-neutral CLI and GitHub Action.
- Linux and macOS support.
- Node.js and Python repository support.
- Git worktree isolation for base and head refs.
- Deterministic proof files validated against `proof.schema.json`.
- Optional Codex evidence summarization that cannot alter verdicts.

## Later

- Additional language adapters.
- Richer report formats.
- Policy packs for organization-specific proof rules.
- Better local developer ergonomics.

## Explicit non-goals for v0.1

- Web dashboard
- Database
- Billing
- Authentication
- Automatic merging
